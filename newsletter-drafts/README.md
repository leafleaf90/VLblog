# Newsletter drafts (repo)

- Add your own `*.md` files here; they are **gitignored** (except this README) so you can keep rough notes and link ideas without shipping them in the repo.
- Copy a starter from `ideas.example.md` if you want a committed template, or create `ideas.md` locally from scratch.
- Link posts with absolute or relative paths from `_posts` when you turn bullets into a send.
